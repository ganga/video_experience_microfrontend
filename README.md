# video_experience_microfrontend

Reusable Flutter package for the live video platform's session lobby, live session shell, replay shell, and failure states.

## Scope

- typed client models for `video-session-service`
- typed HTTP client for session, access-grant, and replay reads
- reusable Flutter screens for:
  - session lobby
  - live session
  - replay
  - access denied
  - replay processing

## Status

This repo is scaffolded locally as the Flutter-first reusable video client for `edtec_mobile`.
